Ввод и вывод данных, оператор присваивания.

Все входные и выходные данные в заданиях этой группы являются вещественными числами.

Begin 1

Дана сторона квадрата a. Найти его периметр P = 4⋅ a.

Код решения
  1. let a = Number(prompt("Введите сторону квадрата"));
  2. let P = 4 * a;
  3. alert("Периметр квадрата: " + P);

Begin 2

Дана сторона квадрата a. Найти его площадь S = a2.

Код решения
  1. let a = Number(prompt("Введите сторону квадрата"));
  2. let S = a * a;
  3. alert("Площадь квадрата: " + S);

Begin 3 Сложный

Даны стороны прямоугольника a и b. Найти его площадь S = a⋅ b и периметр P = 2⋅ (a + b).

Код решения
  1. let a = Number(prompt("Введите сторону 'a' прямоугольника"));
  2. let b = Number(prompt("Введите сторону 'b' прямоугольника"));
  3. let S = a * b;
  4. let P = 2 * (a + b);
  5. alert("Площадь прямоугольника: " + S + "\n" + "Периметр прямоугольника: " + P);

Begin 4

Дан диаметр окружности d. Найти её длину L = π⋅ d. В качестве значения π использовать 3.14.

Код решения
  1. let d = Number(prompt("Введите диаметр окружности"));
  2. let L = 3.14 * d;
  3. alert("Длина окружности: " + L);

Begin 5

Дана длина ребра куба a. Найти объём куба V = a3 и площадь его поверхности S = 6⋅ a2.

Код решения
  1. let a = Number(prompt("Введите длину ребра куба"));
  2. let V = Math.pow(a, 3);
  3. let S = 6 * Math.pow(a, 2);
  4. alert("Объём куба: " + V + "\n" + "Площадь поверхности куба: " + S);

Begin 6

Даны длины рёбер a, b, c прямоугольного параллелепипеда. Найти его объём V = a⋅ b⋅ c и площадь поверхности S = 2⋅ (a⋅ b + b⋅ c + a⋅ c).

Код решения
  1. let a = Number(prompt("Введите длину ребра 'a' параллелепипеда"));
  2. let b = Number(prompt("Введите длину ребра 'b' параллелепипеда"));
  3. let c = Number(prompt("Введите длину ребра 'c' параллелепипеда"));
  4. let V = a * b * c;
  5. let S = 2 * (a * b + b * c + a * c);
  6. alert("Объём параллелепипеда: " + V + "\n" + "Площадь поверхности параллелепипеда: " + S);

Begin 7 Сложный

Найти длину окружности L и площадь круга S заданного радиуса R: L = 2⋅ π⋅ R, S = π⋅ R2.

Код решения
  1. let R = Number(prompt("Введите радиус круга"));
  2. let L = 2 * 3.14 * R;
  3. let S = 3.14 * Math.pow(R, 2);
  4. alert("Длина окружности: " + L + "\n" + "Площадь круга: " + S);

Begin 8

Даны два числа a и b. Найти их среднее арифметическое: (a + b) / 2.

Код решения
  1. let a = Number(prompt("Введите число 'a'"));
  2. let b = Number(prompt("Введите число 'b'"));
  3. let M = (a + b) / 2;
  4. alert("Среднее арифметическое: " + M);

Begin 9

Даны два неотрицательных числа a и b. Найти их среднее геометрическое, то есть квадратный корень из их произведения: √(a * b)

Код решения
  1. let a = Number(prompt("Введите неотрицательное число 'a'"));
  2. let b = Number(prompt("Введите неотрицательное число 'b'"));
  3. let G = Math.sqrt(a * b);
  4. alert("Среднее геометрическое: " + G);

Begin 10

Даны два ненулевых числа. Найти сумму, разность, произведение и частное их квадратов.

Код решения
  1. let a = Number(prompt("Введите первое ненулевое число"));
  2. let b = Number(prompt("Введите второе ненулевое число"));
  3. let Sum = (a * a) + (b * b);
  4. let Dif = (a * a) - (b * b);
  5. let Prd = (a * a) * (b * b);
  6. let Qnt = (a * a) / (b * b);
  7. alert("Сумма квадратов: " + Sum + "\n" + "Разность квадратов: " + Dif + "\n" +
  8. "Произведение квадратов: " + Prd + "\n" + "Частное квадратов: " + Qnt);
